# Clone da interface do app Nubank

Com esse clone foi possível aprender atributos básicos do ConstraintLayout, NestedScrollView, criação de formas e elementos como CardView e RecyclerView.

Achei muito interessante gerar elementos dinâmicamente com o RecyclerView e a implementação do Adapter.
De fato é muito conteúdo para absorver sobre o Recycler, com certeza irei estudar sobre.
